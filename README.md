# 📦 A4S-ERP

Lightweight ERP system for **Stock, Orders, and Warehouse Management**

Built with:

- HTML
- CSS
- Vanilla JavaScript
- Supabase (PostgreSQL)
- GitHub Pages

---

# 🚀 Features

✔ Product Management
✔ Category Management
✔ Warehouse Management
✔ Stock Adjustment
✔ Stock Movement
✔ Purchase Orders (PO)
✔ Sales Orders (SO)
✔ Requisition System
✔ Reports
✔ Database Viewer
✔ User Management

# ⚙️ Setup

1 Open **Settings page**

```
modules/settings/settings.html
```

2 Add Supabase credentials

```
SUPABASE_URL
SUPABASE_ANON_KEY
```

3 Click **Connect**
